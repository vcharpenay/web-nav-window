Web Navigation Window

What is it for?

The Web navigation window is a "window" to the way you navigate on the Web. It is a browser extension that visualizes your browser history as a graph, so that you can see what paths you take on the Web, on which website they start and how they end.

This extension is also part of a research project on hypermedia systems, whose objective is to collect navigation data from human browser histories to compare them with the history of autonoumous Web agents ("bots").

How does it work?

Select a time period (at the bottom of the page) and click "Render". Then, you should see a graph appear. You can zoom in/out and browse the graph.

How to interpret the graph?

This graph is a "navigation graph": every website you've visited is displayed as a node and there is an arc for every link you've clicked. Some of the arcs start and end on the same node. This happens when you navigate to different pages but on the same website.
